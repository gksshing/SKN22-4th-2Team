### 🎨 UI/UX 개발 내용 요약 (MyData & Email 연동)

백엔드에서 확장한 마이데이터 조회 및 이메일 발송 기능을 프론트엔드 화면에 모두 연동했습니다.

1. **마이페이지(히스토리) 고도화 완료**
   - `HistorySidebar` 상단에 검색어 입력창(Keyword)과 위험도/최신순 정렬(SortBy) 드롭다운을 추가했습니다.
   - 사용자가 검색어나 정렬 기준을 바꾸면 백엔드 API (`/api/v1/history?user_id=...&keyword=...&sort_by=...`)로 즉시 요청을 보내 원하는 히스토리만 필터링해서 볼 수 있게 구현했습니다.

2. **분석 결과 이메일 발송 연동 완료**
   - `ResultView` 하단의 '액션 버튼 그룹' 영역에 기존 PDF 캡처 버튼 옆으로 `결과 이메일로 받기` 버튼을 추가했습니다.
   - 버튼 클릭 시 현재 결과를 담아 백엔드의 `/api/v1/notify/email` 로 POST 요청을 보냅니다.
   - 성공/실패 여부에 따라 즉시 사용자에게 브라우저 네이티브 알럿(Toast 대체용 Alert)으로 피드백을 주도록 처리했습니다.
