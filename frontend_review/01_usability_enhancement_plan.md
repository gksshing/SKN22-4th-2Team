### 🛠️ 프론트엔드 코드베이스 분석 결과

현재 `ResultView.tsx` 컴포넌트와 유틸리티 함수들을 분석한 결과는 다음과 같습니다:

1. **'PDF로 다운로드' 버튼 기능 (기구현 완료)**
   - 이미 `html2canvas` 및 `jspdf` 모듈이 설치되어 있으며, `src/utils/exportPdf.ts`를 통해 정상적으로 구현되어 `ResultView.tsx` 레이아웃에 포함되어 있습니다. 
   - 이 부분은 아주 잘 구성되어 있으므로 그대로 사용하면 완벽합니다.

2. **키프리스(KIPRIS) 딥링크 기능 (조건부 수정 필요)**
   - 선행 특허 리스트(`TopPatents`)의 원문 조회 링크는 현재 구현되어 있으나, API에서 넘어오는 데이터 중 `applno`(출원번호)가 존재할 때만 다이렉트 링크가 활성화됩니다.
   - `applno`가 없을 경우에는 KIPRIS의 '범용 메인 검색 페이지 (`#page1`)'로 이동하는 맹점이 존재합니다.
   - **개선 방향:** `applno`가 없을 경우 메인 페이지가 아닌, 특허 번호(`patent.id`)를 사용한 KIPRIS 자체 검색 쿼리 URL(`&searchString=...`)로 이동하도록 수정하겠습니다.
