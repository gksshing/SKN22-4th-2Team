### 🎨 UI/UX 개발 내용 요약
**게스트 접속 환경(Guest Access) 구현**
- **비로그인 홈 화면 진입 허용**: `AuthGuard`의 강제 로그인 로직을 제거하여, 사용자가 앱 접속 즉시 아이디어 검증 기능을 사용할 수 있도록 개선했습니다.
- **선택적 인증 시스템**: 헤더에 '로그인 / 회원가입' 버튼을 추가하고, 클릭 시에만 로그인 폼이 오버레이(Overlay) 형태로 나타나도록 구현했습니다.
- **게스트 히스토리 유지**: 로그인하지 않은 상태에서도 브라우저 세션 ID(`Shortcut-Session-ID`)를 기반으로 최근 분석 이력을 확인할 수 있습니다.
- **인터셉터 수정**: API 요청 시 401(Unauthorized) 에러가 발생해도 강제로 로그인 페이지로 리다이렉트하지 않고, 사용자에게 안내 후 현재 페이지를 유지하도록 조정했습니다.
