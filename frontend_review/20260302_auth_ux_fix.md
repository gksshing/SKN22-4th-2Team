### 🎨 UI/UX 개발 내용 요약
로그인/회원가입 기능을 완전히 복구하고, 사용자 경험을 개선하기 위해 '게스트 모드'와 '헤더 인증 컨트롤'을 구현했습니다.

1. **로그인/회원가입 기능 복구**:
   - 백엔드와 프론트엔드 간의 JWT 쿠키 기반 인증 시스템을 연결했습니다.
   - `auth_router.py`를 메인 API 라우터에 통합하여 모든 인증 엔드포인트를 활성화했습니다.

2. **게스트 모드 (Guest Mode) 구현**:
   - 웹 접속 시 바로 로그인 창이 뜨는 문제를 해결했습니다. 이제 첫 화면에서 메인 페이지가 바로 나타납니다.
   - 비로그인 상태에서도 사이트 구조를 둘러볼 수 있습니다.

3. **인증 요구 로직 최적화**:
   - 특허 검색이나 기록 조회 등 실제 기능 사용 시에만 로그인을 요구하도록 변경했습니다.
   - 로그인 필요 시 알림창과 함께 로그인 모달을 띄워 자연스러운 흐름을 제공합니다.

4. **헤더 인증 컨트롤 추가**:
   - 상단 우측에 로그인, 회원가입 버튼을 배치했습니다.
   - 로그인 성공 시 사용자 이메일 표시와 함께 로그아웃 버튼이 제공됩니다.
