## 🎯 목표

현재 로그인/회원가입 페이지의 어두운 슬레이트 배경(`bg-gradient-to-br from-slate-900`)을 **밝고 깔끔한 화이트 기반 중앙 카드 디자인**으로 리뉴얼합니다. 레퍼런스는 네이버 로그인처럼 **화면 중앙에 심플한 카드 하나**가 있는 구조입니다.

## 🖌️ 디자인 스펙 (Frontend 구현 가이드)

### 1. 배경
- **현재**: `bg-gradient-to-br from-slate-900 to-blue-950` (어두운 그라데이션)
- **변경**: `bg-gray-50` (밝은 연회색) 또는 `bg-white`

### 2. 카드 스타일
| 항목 | 현재 | 변경 |
|---|---|---|
| 카드 배경 | `bg-white` | `bg-white` (유지) |
| 카드 그림자 | `shadow-2xl` | `shadow-md border border-gray-200` (날카롭고 세련되게) |
| 카드 너비 | `max-w-md` | `max-w-md` (유지), 내부 여백 `px-10 py-10` |
| 카드 테두리 | 없음 | `border border-gray-200` (연한 테두리) |

### 3. 브랜드 헤더 (카드 외부 중앙)
```
✂️ Short-Cut         ← 브랜드명 크게 (font-black, text-3xl, text-gray-800)
AI 특허 아이디어 검증  ← 부제목 (text-sm, text-gray-500)
```

### 4. 입력 필드 스타일
- 레이블과 입력창 간 경계를 명확하게: 네이버처럼 라벨이 **필드 위에, 작고 회색**으로
- 입력창: `border border-gray-300 rounded-lg` + focus 시 `border-blue-500 ring-1 ring-blue-100`
- 비밀번호 필드: 오른쪽에 `👁️` 토글 아이콘 추가 (`show/hide password`)

### 5. 주요 CTA 버튼
- **로그인 버튼**: `bg-green-500 hover:bg-green-600 text-white font-bold w-full py-3 rounded-lg`
- 네이버 스타일의 녹색 Primary 버튼 (현재 파란색에서 Green 계열로 변경 또는 유지)
- 로딩 중에는 버튼 내 스피너 표시 (기존 코드 유지)

### 6. 하단 링크 영역
```
계정이 없으신가요?  [회원가입]    |    [비밀번호 찾기]
```
- `text-sm text-gray-500` + 링크는 `text-green-600 font-semibold`

### 7. 에러 메시지
- **인라인 에러**: 각 필드 하단에 `text-red-500 text-xs` (`role="alert"` 유지)
- **서밋 에러**: 버튼 위에 빨간 배경 박스 (기존 코드 유지, 위치만 버튼 위로 이동)

## 🚨 Tech Lead 주의사항

- 접근성(`role="alert"`, `label htmlFor`, `autoComplete`) 속성은 반드시 유지
- 계정 열거 공격 방어용 단일 에러 메시지("아이디 또는 비밀번호를 확인해주세요") 문구는 변경하지 말 것
- 반응형 처리: 모바일에서도 카드가 `px-4` 여백으로 자연스럽게 수축되도록
