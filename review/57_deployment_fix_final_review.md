### 🔍 총평 (Architecture Review)

배포 환경에서 레거시 UI와 목업이 노출되던 서빙 경로 문제를 아키텍처 관점에서 근본적으로 해결했습니다.

- **서빙 안정성**: 백엔드(`main.py`)가 정적 파일 경로를 절대 경로로 탐색하고, 탐색 결과를 명시적으로 로깅하도록 개선되어 운영 환경의 불확실성을 제거했습니다.
- **빌드 무결성**: Docker 빌드 프로세스에서 기존 에셋을 명시적으로 정리(`rm -rf dist`)하고 최신 React 결과물만 복사하도록 강제하여 버전 정합성 문제를 원천 차단했습니다.
- **메인 페이지 접근**: 루트 경로(`/`)가 정상적으로 최신 `index.html`을 반환하고 이어서 React SPA가 마운트되는 구조를 확보했으므로, 특허 분석 메인 페이지가 정상 노출될 것으로 확신합니다.
